# PDFx Merge Tool

A PDF Merge Web Application built using Node.js, Express.js, HTML, CSS and JavaScript.

## Features
- Upload Multiple PDF Files
- Merge PDFs Instantly
- Download Merged PDF
- Responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- PDF-Lib
